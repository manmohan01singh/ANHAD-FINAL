🎯 Gurpurab Calendar Fixes

Production-ready fixes for scroll issues and event semantics

🔴 Issue 1: UI Disappearing on Scroll FIXED
Root Cause Identified
The calendar was using content-visibility: auto and contain-intrinsic-size which caused rendering glitches during scroll. These CSS properties are meant for performance but can cause elements to disappear/reappear unpredictably.
/* BEFORE (Problematic) */ .event-row { content-visibility: auto; contain-intrinsic-size: auto 52px; will-change: scroll-position; } /* AFTER (Fixed) */ .event-row { position: relative; backface-visibility: hidden; -webkit-backface-visibility: hidden; }
Additional Optimizations
  • Removed aggressive contain properties from page container
  • Added backface-visibility: hidden for GPU acceleration
  • Removed will-change: scroll-position (causes repaints)
  • Ensured stable rendering with transform: translateZ(0)
🟠 Issue 2: Event Type Semantics FIXED
Intelligent Event Classification
Implemented a smart classification system that checks both event type AND event name to properly categorize events as memorial (Jyoti Jyot) or celebration (Prakash, Gurgaddi).
function getEventCategory(type, eventName = '') { const name = String(eventName || '').toLowerCase(); // PRIORITY 1: Check event name for explicit memorial indicators if (name.includes('jyoti jot') || name.includes('joti jot') || name.includes('ਜੋਤੀ ਜੋਤ') || name.includes('shaheedi') || name.includes('ਸ਼ਹੀਦੀ') || name.includes('barsi')) { return 'memorial'; } // PRIORITY 2: Check type for memorial events const memorialTypes = ['shaheedi', 'historical', 'joti-jot', 'jyoti-jot', 'barsi']; if (memorialTypes.includes(String(type).toLowerCase())) { return 'memorial'; } // PRIORITY 3: Check for celebration events const celebrationTypes = ['prakash', 'gurgaddi', 'janam', 'vaisakhi', 'dastar']; if (celebrationTypes.includes(String(type).toLowerCase()) || name.includes('prakash') || name.includes('gurgaddi') || name.includes('ਪ੍ਰਕਾਸ਼') || name.includes('ਗੁਰਗੱਦੀ') || name.includes('vaisakhi') || name.includes('ਵੈਸਾਖੀ')) { return 'celebration'; } return 'neutral'; }
