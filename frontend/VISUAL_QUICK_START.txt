┌──────────────────────────────────────────────────────────────────────────┐
│  TROUBLESHOOTING                                                         │
└──────────────────────────────────────────────────────────────────────────┘

    ❌ Python not found?
       → Download from: https://www.python.org/downloads/
       → Install with "Add Python to PATH" checked
       → Restart Command Prompt

    ❌ Pillow not installed?
       → The batch file will install it automatically!
       → If it fails: pip install Pillow

    ❌ Logo looks blurry?
       → Check source logo is at least 512×512 pixels
       → Ensure it has transparent background
       → Re-run generation script

    ❌ Files not created?
       → Check you're in the frontend/ folder
       → Verify newlogo-removebg-preview.png exists
       → Run verify-logos.py for details
