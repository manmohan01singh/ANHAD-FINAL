## 🚀 HOW TO USE

### Voice Search
1. Click the microphone button in search bar
2. Speak your search query in Punjabi/English
3. Watch it transcribe in real-time
4. Results appear automatically

### Text Search
1. Type in the search bar
2. Choose search type (First Letter, Gurmukhi, English, Ang)
3. Results appear as you type (debounced)
4. Click any result to view full Shabad

### Gurmukhi Keyboard
1. Click the "ੳ" button in search bar
2. Tap letters to compose
3. Use matras for vowel marks
4. Click search icon or "Done"

### Ang Navigation
1. Use +/- buttons or slider
2. Enter number directly (1-1430)
3. Click "Open Ang"
4. View all lines from that page

### Quick Searches
1. Tap any popular search card
2. Instantly see results
3. No typing needed
